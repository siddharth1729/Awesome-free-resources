Q 1 - In which of the Production type, produce materials changed with each lot and costs are calculated as per orders and lots?

A - Repetitive production

B - Discrete Production

C - Manufacturing Production

D - Material Requirement Planning

Answer : B

Explanation

Discrete production is known as industry where produce materials changed with each lot and costs are calculated as per orders and lots.

In Repetitive production, product is not changed for longer time period and production is done as total quantity and not in the form of individual lots.

Q 2 - When you create a Material BOM without a Plant is known as?

A - Reference BOM

B - Master BOM

C - Group BOM

D - Master group

Answer : C

Q 3 - To create routing it is mandate that you first define work center?

A - True

B - False

Answer : A

Q 4 - When you use Planning strategy 10, which of the following is correct?

A - Only PIR quantity is considered for MRP run & PIR requirement type is VSF.

B - Sales order are completely ignored.

C - For MRP run maximum of 2 PIR and Sales order can be considered

D - None of the above

Answer : A

Explanation

When you use Planning strategy 10, only PIR quantity is considered for MRP run and sales order are completely ignored. In this PIR requirement type LSF and is reduced when you deliver stock to customer.

Q 5 - Which of the following doesn’t represent an issue Material Requirement Planning element?

A - Sales Order

B - PIR

C - Transfer Order

D - Production Order

Answer : D

Explanation

Issues include PIRs, Dependent Requirements, Order Reservations, Sales Orders, Issuing Stock Transfer Orders, etc. and receipts include Production Orders, Purchase Requisitions, Purchase Orders, Open Production Orders, Receiving Stock Transfer Order, Schedule lines, etc.

Q 6 - Which of the following is performed using Planning table and orders are processed as define in work center to meet the Production process?

A - Capacity Planning

B - Capacity Leveling

C - Capacity Requirement gathering

D - All of these

Answer : B

Explanation

Capacity leveling is performed for performing detailed production planning purpose. This is done through planning table which is used to carry detailed planning of capacity requirements over time in future.

Q 7 - NETCH is used in?

A - Work Center

B - MRP Processing Key

C - BOM

D - Production Order

Answer : B

Explanation

NetChange in planning horizon −

This run considers the requirement in predefined planning horizon and not like net change which check the future requirements.

In this run, system consider only those material which has undergone some changes from their last MRP run.

Q 8 - When material is produced as per Production order and goods is placed at storage location, which of the following is correct?

A - Stock Quantity is increased and movement type is 103

B - Stock Quantity remain same and movement type is 103

C - Stock Quantity is increased and movement type is 101

D - Stock Quantity remains same and movement type is 101

Answer : C

Explanation

Good receipt is performed when material is produced as per Production order and goods is placed at storage location. The stock quantity is increased and movement type 101 is entered.

Q 9 - Which of the following is correct about regenerative planning and net change planning?

A - Planning is performed online however in NETCH it is performed as background job.

B - Planning of material is as per demand, in other one it is based on previous productions.

C - Planning all material from planning file, you can plan for material for which planning file exists.

D - All of these

Answer : C

Q 10 - To set your PIR version active, which of the below value should be used to specify requirement would be considered in material requirement planning?

A - 00

B - 01

C - 02

D - 03

Answer : A

Explanation

In SAP PP, demand management is performed by Planned Independent requirement PIR. Planned Independent requirement provides input for production planning. A PIR contains one planned quantity of product and one date for material or a planned quantity is split over a span of time period.

To set your PIR version active use value 00 to specify requirement would be considered in material requirement planning. To maintain number of versions of planned independent requirement, you can set some numbers to active and others to inactive.
