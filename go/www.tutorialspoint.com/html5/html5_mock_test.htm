Q 1 - Which of the following is true about HTML 5?

A - HTML5 is the next major revision of the HTML standard superseding HTML 4.01, XHTML 1.0, and XHTML 1.1.

B - HTML5 is a standard for structuring and presenting content on the World Wide Web.

C - HTML5 is a cooperation between the World Wide Web Consortium (W3C) and the Web Hypertext Application Technology Working Group (WHATWG).

D - All of the above.

Answer : D

Explanation

All of the above options are correct.

Explanation

All of the above options are new features introduced in HTML5.

A - true

B - false

Answer : A

Explanation

A - true

B - false

Answer : B

Explanation

Q 9 - Which of the following tag represents a generic document or application section in HTML5?

A - section

B - article

C - aside

D - header

Answer : A

Explanation

'section' tag represents a generic document or application section. It can be used together with h1-h6 to indicate the document structure.

C - aside

D - header
