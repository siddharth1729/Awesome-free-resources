Os desenvolvedores Melhores Práticas Tutorial

Este pequeno tutorial é baseado no meu passado MAIS DE 16 anos de experiência no setor de desenvolvimento de software. Tenho passado por diversas fases de minha carreira a partir do formando até desenvolvedor de software gerenciamento sênior.

Não quero manter o meu aprendizado para mim, assim que eu havia escrito um pequeno tutorial há alguns anos, e depois de obter muito de motivação dos meus queridos leitores, eu pensei em rever e adicionar mais alguns aprendizados que podem beneficiar muitos outros engenheiros de software e desenvolvedores que trabalham com esta adorável indústria.

Não vou ditar qualquer um dos pontos, mas todas as práticas listadas aqui contribuíram muito na minha carreira, desenvolvimento de software, portanto se você acha que eles fazem algum sentido para você, em seguida, tente adoptar alguns. Se você tiver qualquer +/- comentários, por favor sinta-se livre para escrever-me de volta : Entre em Contato Conosco

Audiência

Se você estiver trabalhando para indústria de software como um engenheiro de software ou um desenvolvedor de software, então eu tenho certeza de que vão gostar deste tutorial. Tente relacionar os factos mencionados no tutorial com o seu dia-2-dia e encontrar tantos fatos ocultos, que são, de facto, muito óbvio, mas nós nunca se deu nossa atenção para elas.

Pré-requisitos

Antes de gravar todas as práticas mencionadas neste pequeno tutorial, fiz uma suposição de que você está trabalhando como um software profissional e você entender software básico terminologias e a atmosfera em torno de um software profissional.
