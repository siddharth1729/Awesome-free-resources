Guida HTML

HTML significa letteralmente Hyper Text Markup Language, ed è il linguaggio Web più diffuso per lo sviluppo di pagine internet

HTML è stato creato da Berners-Lee nel lontano 1991, le prime specifiche dello stardard HTML furono pubblicate nel 1995 con "HTML 2.0". La versione principale di HTML è stata HTML 4.01, pubblicata alla fine del 1999. Anche se la versione HTML 4.01 è ancora ampiamente utilizzata, attualmente è in uso HTML-5, pubblicata nel 2012 ed è un'estensione di HTML 4.01.

Audience

Questo tutorial è rivolto agli aspiranti Web designer e Sviluppatori che necessitano di comprendere l'HTML nel suo insieme con una semplice panoramica e degli esempi pratici. Verranno fornite le conoscenze per iniziare ad utilizzare l' HTML e puntare ad un livello di competenza più alto.

Prerequisiti

Prima di procedere con questo tutorial dovreste avere una conoscenza basilare dei sistemi operativi Windows o Linux. Inoltre dovrete avere alcune conoscenze base come:

  • Esperienza con qualsiasi editor di testo come notepad, notepad++, o Editplus ecc.
  • Capacità di creare cartelle e file nel vostro computer.
  • Capacità di spostarsi all'interno delle cartelle.
  • Capacità di scrivere all'interno di un file e salvarlo.
  • Conoscere i vari formati delle immagini come JPEG e PNG.

Prova HTML Online

Per molti degli esempi forniti in questo tutorial troverai l’opzione Try it che permette di mostrare l'output del vostro codice facilitando quindi l'apprendimento.

Prova l'esempio seguente tramite il pulsantino Try it che si trova nell'angolo in alto a destra del box contenente il codice −

<!DOCTYPE html>
<html>
   <body>
      <h1>Hello World!</h1>
   </body>
</html>
