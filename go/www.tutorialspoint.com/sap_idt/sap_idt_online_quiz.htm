Following quiz provides Multiple Choice Questions (MCQs) related to SAP IDT. You will have to read all the given answers and click over the correct answer. If you are not sure about the answer then you can check the answer using Show Answer button. You can use Next Quiz button to check new set of questions in the quiz.

Q 1 - Which of the following allows you to create a data model as multisource enabled that extracts data from one or more data sources using multiple connections?

A - IDT

B - UDT

C - CMC

D - Explorer

Answer : A

Explanation

IDT is multi source enabled means you can extract the data from different data sources while creating a Universe.

Q 2 - You want to create a Relational connection in IDT that points to SAP HANA database, which of the following database middleware drivers you should select?

A - SAP ERP 6

B - MaxDB 7.7

C - SAP HANA database 1.0

D - mySAP ERP 2004

Answer : C

Q 3 - While defining a connection in IDT, which of the following is not an option under connection parameters?

A - Pool time Out

B - Array fetch size

C - Array Bind size

D - Login Timeout

E - Authentication Mode

Answer : E

Explanation

You can define connection parameters like - Time out, Array fetch size, Array Bind size, etc.

Q 4 - Which of the following can be used to break the loops in Data Foundation when you are not able break loop using an Alias table?

A - Alias Table

B - Derived Table

C - Context

D - Joins

Answer : C

Explanation

Context are also used to break the loops in Data Foundation when you are not able to break loop using an Alias table.

When you have multiple fact tables that are connected to a single dimension table, you create a Context for each fact table in this case.

Q 5 - Which of the following feature can be used to facilitate custom drill down between objects from same or different classes according to user requirement?

A - Custom Hierarchies

B - Linked Universe

C - List of Values

D - Aggregate Awareness

Answer : A

Q 6 - To use same font for all the components, which of the following font type should be selected under document properties?

A - Global Font

B - Device Font

C - Embedded Font

D - None of these

Answer : A

Explanation

Different fonts can be set for different components. There is an option to setup the global font if you want to use same font for all the components. This can be used on device fonts or on embedded fonts. When you use machine fonts, exported file is smaller and you can’t rotate text. When embedded fonts are used, you can rotate text and file size is large.

Q 7 - You want to specify currency symbol to show before the value, which of the following component property should be used?

A - General

B - Behavior

C - Appearance

D - Alerts

Answer : C

Explanation

Go to Properties → Appearance → Text

Q 8 - In SAP Dashboards, you can apply same layout, appearance and formatting to data models when you need all models with same look?

A - True

B - False

Answer : A

Explanation

Using themes, you can apply same layout, appearance and formatting to data models when you need all models with same look. Theme provides you with a color scheme but you can customize it as per requirement.

To apply a theme, go to Format → Theme

Q 9 - What is the maximum number of rows that you can embed to a spreadsheet to use as component binding?

A - 256

B - 512

C - 1024

Answer : B

Explanation

You can select a maximum range of 512 rows in range. Larger the value slower the performance of data model. To change this value, go to File → Preferences → Excel Options.

Q 10 - Using ignore excel formula error options, you can make cells that contain invalid excel formulas as empty cells under Excel option?

A - True

B - False

Answer : A
