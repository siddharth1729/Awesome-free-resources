Quantitative Aptitude Tutorial

An aptitude test represents a sytematic way to judge a candidate's mental capability as how he/she performs certain tasks and reacts to different scenarios. Each aptitude test uses a standard method of administration and scoring so that results can be computed and compared with the other candidates.

This tutorial will teach you basic concepts of aptitude like numerical computation ability, analytical abilities. The tutorial is divided into sections such as Number System, Basic Arithmetic, Decimals and so on. Each of these sections contain related topics with simple and useful examples.

Audience

This reference has been prepared for the candidates to help them to understand the basic to advanced concepts required to pass the Aptitude Tests, IAS, PCS and other Civil Services Exams. This tutorial will give you enough understanding on various concepts where you can take yourself to a higher level of expertise.

Prerequisites

Before proceeding with this tutorial you should have basic knowledge of Mathematics.
