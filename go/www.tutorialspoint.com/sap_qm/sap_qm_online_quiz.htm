Following quiz provides Multiple Choice Questions (MCQs) related to SAP QM. You will have to read all the given answers and click over the correct answer. If you are not sure about the answer then you can check the answer using Show Answer button. You can use Next Quiz button to check new set of questions in the quiz.

Q 1 - Under quality control, you have quality notifications, standard reports and Quality notification system. What actions has to be taken as per defect finding?

A - True

B - False

Answer : A

Q 2 - Which of the following module is integrated with QM to manage the costs for defect control and non-confirmative costs in production process?

A - SAP Material Management

B - SAP Production Planning

C - SAP Sales and Distribution

D - SAP Controlling

Answer : D

Explanation

You can integrate SAP QM with controlling process to manage the costs for defect control and non-confirmative costs in production process.

Q 3 - A sampling plan defines the sample size specification for the inspection lot to determine if a sample is accepted or not accepted?

A - True

B - False

Answer : A

Q 4 - Which of the following is a charactertics of quality control chart type?

A - The characteristics for which the control chart is suited

B - The control variables a chart contains

C - How the control limits are calculated

D - All of the above

Answer : D

Q 5 - What do you understand by AQL?

A - Acceptable Quality Limit

B - Automatic Quality Length

C - Acceptable Quality Length

D - Automatic Quality Limit

Answer : A

Explanation

AQL (Acceptable Quality Level) is the quality level that represents the upper limit of a satisfactory average quality level for an acceptance sampling inspection.

Q 6 - What is SAP inspection type used for in process material inspection?

A - 01

B - 02

C - 03

D - 04

Answer : C

Q 7 - What is the standard SAP QM notification type to record a complaint against vendor and take action for the incoming material?

A - Q1

B - Q2

C - Q3

D - Not Defined

Answer : B

Explanation

You can use standard SAP QM notification type Q2 to record a complaint against vendor and take action for the incoming material.

Q 8 - For each good receipt, SAP system creates a one inspection lot which contains details- vendor, date or purchase, etc.?

A - True

B - False

Answer : A

Q 9 - If you want to display Quality Management standard forms, which of the following can be used?

A - QCCF

B - QCCM

C - QCCN

D - None of the above

Answer : A

Q 10 - Which of the following module is integrated with QM to manage the costs for defect control and non-confirmative costs in production process?

A - SAP Material Management

B - SAP Production Planning

C - SAP Sales and Distribution

D - SAP Controlling

Answer : D

Explanation
