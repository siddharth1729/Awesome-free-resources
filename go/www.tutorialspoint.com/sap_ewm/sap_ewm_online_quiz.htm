Following quiz provides Multiple Choice Questions (MCQs) related to SAP EWM. You will have to read all the given answers and click over the correct answer. If you are not sure about the answer then you can check the answer using Show Answer button. You can use Next Quiz button to check new set of questions in the quiz.

Q 1 - Which of the following is a type of Storage Section?

A - Fast Moving

B - Open Storage

C - Picking

D - All of the above

Answer : A

Explanation

Storage Section

This is part of storage type and represents a group of bins with same characteristics. Commonly used storage sections are fast moving or slow moving items, etc.

Q 2 - Which of the following area of Storage type is used where products are labeled, identified or cross checked during goods receipt process?

A - Pick Point

B - Identification Point

C - Work Center

D - Yards

Answer : B

Explanation

Identification Point

An area where products are labeled, identified or cross checked during goods receipt process.

Q 3 - Which of the following correctly defines difference between Storage unit and Handling Unit?

A - Storage unit is a container in which a material stored or transported and handling unit is combination of material and packaging material.

B - Handling unit is a container in which a material stored or transported and Storage unit is combination of material and packaging material.

C - Both are same

Answer : A

Q 4 - Rack, Open, Shelf, Damage represents the?

A - Stock Types

B - Storage Types

C - Plant Type

D - None

Answer : B

Explanation

Stock Types − Unrestricted, Blocked, Quality

Storage Types − Rack, Open, Shelf, Damage

Q 5 - When an outbound delivery request is created, EWM system doesn’t create a warehouse task or task is created with a partial quantity, this is known as?

A - Stock removal strategy

B - Denial Scenario

C - Goods Issue

D - None of the above

Answer : B

Explanation

When an outbound delivery request is created, EWM system doesn’t create a warehouse task or task is created with a partial quantity, this is called Denial scenario.

Q 6 - Combining warehouse request items and divide them as per possible criteria like activity area, route or product is known as?

A - Stock Division in Warehouse

B - Waves in Warehouse

C - Stock Removal in Warehouse

D - None of the above

Answer : B

Explanation

In SAP EWM, you can combine warehouse request items and divide them into waves as per possible criteria like activity area, route or product. Waves can be created automatically in SAP extended warehouse management system or manually using defined waves templates.

Q 7 - Which of the following is used to execute put away or stock removal process in warehouse?

A - Storage Control

B - Stock Denial

C - Stock Removal

D - Outbound Delivery

Answer : A

Explanation

Storage control is used to determine product movements in warehouse. It is used to execute put away or stock removal process in warehouse.

These activities are performed as per the physical structure of warehouse.

Q 8 - Arrange the following documents in correct order in which they are created?

1. ERP: Purchase Order

2. EWM: Inbound Delivery

3. EWM: Inbound Delivery Notification

4. ERP: Inbound Delivery

A - 1,2,3,4

B - 1,3,4,2

C - 1,4,3,2

D - 1,4,2,3

Answer : C

Explanation

ERP: Advanced shipping notification/Purchase order

ERP: Inbound delivery (ID)

EWM: Inbound delivery notification (IDN)

EWM: Inbound delivery

Q 9 - The main difference between the planned and the automatic replenishment method is that the automatic method is executed during task confirmation?

A - True

B - False

Answer : A

Q 10 - Time to perform Warehouse activities like picking, packing, put away, etc. is called?

A - Direct Labor

B - Indirect labor

C - Unproductive Labor

D - None of the above

Answer : A

Explanation

  • Direct Labor − This includes time to perform warehouse activities like picking, packing, put away, etc.

  • Indirect Labor − This includes extra activities to manage warehouse like Sweeping, cleaning, housekeeping etc.

  • Unproductive time − Breaks, etc.
