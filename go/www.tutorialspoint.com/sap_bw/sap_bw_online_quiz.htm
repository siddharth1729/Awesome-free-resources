Following quiz provides Multiple Choice Questions (MCQs) related to SAP BW. You will have to read all the given answers and click over the correct answer. If you are not sure about the answer then you can check the answer using Show Answer button. You can use Next Quiz button to check new set of questions in the quiz.

Q 1 - Which of the following transaction can be used to adjust an infoset after making changes

A - RSISET

B - RSKC

C - RRBS

D - none of the above

Answer : A

Explanation

RSISET transaction can be used to adjust changes in an infoset.

Q 2 - Which of the following component in BW architecture can be used for detailed storage of data and used for BEx reporting?

A - PSA

B - Operational Data Store

C - InfoCubes

D - Infoarea

Answer : B

Explanation

In SAP BW architecture, ODS can be used for detailed storage of data and used for BEx reporting. It exists between PSA and InfoCubes.

Q 3 - In an Operational Data store, how many number of maximum keys an object can have?

A - 8

B - 16

C - 24

D - 32

Answer : B

Q 4 - Which of the following transaction code can be used for monitoring daily process chains?

A - RSPC1

B - RSPCM

C - RSPC

D - RZ20

Answer : B

Explanation

RSPC1: Display Process Chain

RSPCM: Monitoring Daily Process Chains

RSPC: Process Chain Maintenance

RZ20: Log for Process Chains

Q 5 - In SAP BW, which of the following transaction can be used to display tables in Star schema?

A - DISPLAYSCHEMA

B - VIEWSCHEMA

C - LISTSCHEMA

D - All of the above

Answer : C

Q 6 - In SAP BW, to create new InfoObjects to store data, which of the following tab is used in Administration workbench?

A - Administration

B - Data Source

C - Modeling

D - Translation

Answer : C

Explanation

In RSA1, you use Modeling tab to create new objects to store BI data.

Q 7 - In a Log view under process change management, a yellow mark shows that a process chain

A - Not Yet Run

B - Completed with no errors

C - Running

D - Failed

Answer : C

Explanation

  • Grey: Not Yet Run
  • Green: Completed with no errors
  • Yellow: Running
  • Red: Failed

Q 8 - Which of the following defines the use of InfoSet?

A - When a Join is required

B - When a Union is required

C - Both

D - None of the above

Answer : A

Q 9 - Which of the following in BEx suite is used to create BI queries?

A - Query Designer

B - Report Designer

C - Web Application Designer

D - Web Analyzer

Answer : A

Explanation

Query Designer in BEx suite is used to create BI queries and is user friendly tool.

Q 10 - Which of the following class contains majority of authorization objects in BI?

A - AS

B - DS

C - RS

D - ZS

Answer : C

Explanation

Authorization objects are grouped according to authorization object classes and majority of objects are in class RS in BI.
