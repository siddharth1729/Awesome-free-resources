SAP BW Mock Test I

Q 1 - Which statement(s) is true about Multiproviders?

A - Virtual Infoprovider in SAP BW stores the data.

B - Multiprovider consists of InfoCubes, info objects and info sets

C - More than one info provider is required to build a Multiprovider

D - It is similar to joining the data tables

Answer : B

Explanation

To build MultiProvider, you have to use InfoCubes, info objects and info sets

Q 2 - To use special characters in BW, which of the following transaction can be used?

A - RSA1

B - RSA15

C - RSKC

D - RRBS

Answer : C

Explanation

RSKC is used to enter special characters in BW.

Q 3 - In SAP BW, which of the following is incorrect about flat file upload?

A - Table to store flat file upload parameters is RSADMINC

B - You can upload csv files

C - To set parameters for flat file load, use transaction RSCUSTV7

D - All of the above

Answer : C

Explanation

To set flat file parameters, you can use transaction RSCUSTV7

Q 4 - When you create a dimension as line item dimension in an infocube, ID's and SID's are same in this case?

A - True

B - False

Answer : A

Explanation

SID's of dimension are stored in fact tables and can be used as dimension id's.

Q 5 - Which of the following transaction can be used to adjust an infoset after making changes

A - RSISET

B - RSKC

C - RRBS

D - none of the above

Answer : A

Explanation

RSISET transaction can be used to adjust changes in an infoset.

Q 6 - Which of the following transactions can be used for maintaining data sources in BW?

A - RSA1

B - RSA3

C - RSA5

D - RSA6

Answer : D

Explanation

RSA1 is used to open administration workbench.

RSA3 is used for checking data sources for extraction

RSA6 is used for mainting data sources

RSA5 is used for installing buisness content data sources

Q 7 - What do you understand by data integrity in database?

A - Eliminating incorrect data

B - Eliminating duplicate data

C - Eliminating errors in data transformation

D - All of the above

Answer : A

Explanation

Data integrity deals with removing duplicate data in database.

Q 8 - Which of the following component in BW architecture can be used for detailed storage of data and used for BEx reporting?

A - PSA

B - Operational Data Store

C - InfoCubes

D - Infoarea

Answer : B

Explanation

In SAP BW architecture, ODS can be used for detailed storage of data and used for BEx reporting. It exists between PSA and InfoCubes.

Q 9 - Which of the following is/are incorrect about Infocubes?

A - InfoCubes doesn't contain a key.

B - InfoCubes contain flat file structure & An Infocube normally contains detailed level data..

C - An Infocube can contain data from two or more Operational data stores.

D - None

Answer : B

Explanation

InfoCubes doesn't contain a key.

InfoCubes contain star schema structure.

Infocube contains refined data and not detailed data.

An InfoCube can contain two or more operational data stores.

Q 10 - Which of the following is used for dividing table on row or column based and is used for report performance enhancement?

A - Partitioning

B - Reconstruction

C - Indexing

D - Roll up

Answer : A

Explanation

Partioning is defined as dividing table based on available fields and it is easier to perform a search in smaller tables and hence reporting performance is enhanced.

Q 11 - In an Operational Data store, how many number of maximum keys an object can have?

A - 8

B - 16

C - 24

D - 32

Answer : B

Q 12 - When you upload the same data again in an Operational data store in overwrite mode, it creates new entries in change log.

A - True

B - False

Answer : B
